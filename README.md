# Flappy-bird-javafxx
A Flappy Bird clone built in java using Java-FX with real time physics and collision detection

## Features
- Gravity and jump mechanics
- Obstacle collision detection
- Game over screen
- Keyboard-controlled gameplay

## Tech Used
- Java
- JavaFX

## Requirements
- JDK [your version here]
- JavaFX compatible with the installed JDK

## How to Run
1. Open the project in your Java IDE
2. Set the project SDK to the correct JDK version
3. Run the main file

## Current Status
This project is functional and playable. Future improvements include score tracking and automatic game restart.

## What I Learned
This project helped me practice object-oriented programming, event handling, game loops, and collision detection.
